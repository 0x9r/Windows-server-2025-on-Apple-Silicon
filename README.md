# Windows-server-2025-on-Apple-Silicon
In this repository, I am going to show you how to virtualize Windows Server 2025 on Apple Silicon (ARM64) processors.

<img src="https://github.com/AnderMendoza/AnderMendoza/raw/main/assets/line-neon.gif" width="100%">


The first step will be to create the ISO from UUP Dump (https://uupdump.net/known.php?q=windows+server+arm64). We need to install it precisely. I recommend installing it on Windows since it has given me quite a few problems on macOS.
